# sml
# 科学メディアリテラシー探偵団 ─ 先生用ガイド

中学生向けの科学メディアリテラシー（SML）デジタル教材です。
SNS風のニセ科学ニュースを「先に」体験させることで、実際の誤情報への免疫をつくる**プレバンキング型**の教材です。

---

## 1. この教材で学べること

### （1）情報をチェックする2つの「技」
- **縦読み** … 情報を最後まで確認する読み方。「…更に表示」を開き、コメント欄まで読み切る。最後まで確認せずにシェアしない。最後まで読む中で怪しいポイントが見つかることも多い。
- **横読み** … 今のサイトを離れ、新しいタブで検索する読み方。情報源（だれが言っているか）や、他のサイトに同じ話題があるかを確かめる。

### （2）信頼性を評価する20のチェックポイント
記事の怪しい箇所に**電球**が点灯します。電球の色は評価の観点（What/Who/How）を表します。

| 電球の色 | 分類 | チェックすること | 例 |
|---|---|---|---|
| 🔴 赤 | **What** | 研究内容は確かか | 条件制御・データの数・新薬開発の過程・因果と相関・チェリーピッキング・研究手法 |
| 🟢 緑 | **Who** | どの科学者・発信者が信頼できるか | 研究者の実績・専門性・利益相反・科学的コンセンサス・公的機関・マスメディア・科学研究費・企業の利益 |
| 🟡 黄 | **How** | ネットの表現と人の受け止め方 | 拡散希望・タイトルの誇張・科学イメージ・グラフのわな・いいねの数・エコーチェンバー |

### （3）教材の構成（6事例）
- **縦読み教材**：がん治療（成分Pでがん撲滅）／納豆ダイエット／ノーベル賞とチョコ
- **横読み教材**：地球温暖化（IPCC・IEA）／たばことがん（利益相反）／ワクチンと不妊
- 練習用：アロンアルファ止血（縦読み）／バナナの遺伝子（横読み・検索結果一覧つき）

各電球にはクイズ（はい・いいえの2択）→漫画つき解説→チャレンジ問題（4択）があり、**2問正解で消灯、1問で半灯**。1問10点、全40問で**400点満点**。教材ごとに学習ランクバッジ（🥇満点／🥈得点あり／🥉全問回答）が付きます。

## 2. ファイルと起動方法

| ファイル | 用途 |
|---|---|
| `科学メディアリテラシー探偵団.html` | 通常版 |
| `科学メディアリテラシー探偵団_プレポスト版.html` | 効果測定用。**起動直後に事前テスト（12問）**→全教材終了後に**事後テスト** |
| `科学メディアリテラシー探偵団_ゲーム版.html` | **ゲーム性強化版**（家庭学習・自由学習向け）。学習内容・得点は通常版と同一 |
| `学習ワークシート.docx` | 授業用ワークシート（印刷して配布） |
| `振り返りシート.docx` | 授業末の振り返り用（印刷して配布） |
| `学習ワークシート（ゲーム版）.docx` | ゲーム版用「捜査ワークシート」（事件File・相棒・獲得バッジ欄つき） |
| `振り返りシート（ゲーム版）.docx` | ゲーム版用「捜査報告書」（探偵ランク・実績・最高コンボの記録欄つき） |

**HTMLファイルをダブルクリックするだけで動きます**（インターネット・サーバー不要）。
⚠️ HTMLと `pic` フォルダは同じ場所に置いたままにしてください。

### ゲーム版の追加要素

学習内容・クイズ・得点（400点満点）は通常版と同じで、以下のゲーム要素が加わります。
- **ストーリー**：探偵団の新人として「依頼書」を受け取り、6つの「事件File」を捜査する設定
- **探偵ランク**：正解数に応じて 🔰見習い→🕵️新米→🎩一人前→🧠敏腕→🏆マスター→👑伝説の探偵 にランクアップ（演出つき）
- **コンボ**：連続正解で「🔥◯COMBO!」表示
- **実績コレクション**：「はじめての消灯」「縦読みマスター」「パーフェクト事件」など10種の実績バッジ
- **演出・効果音**：正解音・紙ふぶき・「事件解決！！」画面（あばいた「だましのテクニック」一覧つき）。効果音はヘッダーの🔊ボタンでオフにできます（教室では🔇推奨）
- 学習記録は通常版とは別に保存されます（ログのダウンロード・インポートは共通機能）

## 3. 授業での使い方（推奨50分×1〜2コマ）

1. **導入**：「①チェックの技を練習する！」で縦読み・横読みを全員で練習（10分）
2. **「②学び方を学ぶ！」**で電球の色（What/Who/How）と得点の仕組みを確認（5分）
3. **「③6つの教材にチャレンジ！」**：縦読み3教材→横読み3教材の順に取り組む（25〜60分）
4. **振り返り**：図鑑で分類ごとの得点を確認し、振り返りシートを記入（10分）

### 対話を重視した活用（推奨）
- **2名で1台の端末**を使います。
- **電球が現れたら、クイズに答える前に「なぜここに電球がついているのか」を2人で話し合わせて**から回答させます（電球の位置＝記事の怪しい箇所、色＝評価の観点です）。
- 話し合った内容は**学習ワークシート**にメモさせ、授業の最後に**振り返りシート**を書かせます。
- 個人（家庭）学習でもそのまま使えます。その場合は「じっくり考えてから答える」よう声かけしてください。

### 効果測定（研究・評価用）
プレポスト版では、同一の12問を事前・事後に出題し、結果を比較表示します（各テスト1回のみ受験可）。

## 4. 学習記録

「📊きろく」画面から：
- **学習ログをダウンロード（JSON）** … 家庭や次時に続きから再開する用。「インポート」で読み込み。
- **回答データをCSVで出力** … 全操作・全回答の時系列データ（Excelで開けます）。クラスの分析用。
- ブラウザにも自動保存されます（同じPC・同じブラウザなら自動継続）。

## 5. 補足（元データとの関係）

- クイズ・解説はExcel「各教材に埋め込まれた信頼性評価規準.xlsx」、画面設計は「アプリイメージ.pptx」に基づきます。
- 漫画・写真は `pic` フォルダを参照します（同名で差し替え可能）。画像ファイル名はWeb公開時の文字化けを防ぐため**半角小文字**に統一しています。主な対応：
  - 記事内の写真：`kenkyusha.png`（研究者イメージ）／`gonzalez.png`（ゴンザレス教授）／`banana.png`（バナナシール）／`resultofnattodiet.png`（納豆グラフ）／`bulbcolor.jpg`（電球色の図）
  - 漫画：`kakusan`（拡散希望）・`kocho`（タイトルの誇張）・`shinyaku`（新薬の開発）・`kakenhi`（科学研究費）・`science`（科学イメージ）・`joken`（条件制御）・`rct`（ランダム比較二重盲検）・`data`（データ数）・`graph`（グラフのわな）・`iine`（いいねの数）・`inga`（因果と相関）・`rieki`（発信者の利益）・`media`（マスメディア）・`consensus`（科学的コンセンサス）・`cherry`（チェリーピッキング）・`kokyo`（公的機関の報告書）・`jisseki`（研究者の実績）・`senmon`（研究者の専門性）・`coi`（利益相反）・`echo`（エコーチェンバー）・`method`（信頼のある研究手法）※すべて.png
